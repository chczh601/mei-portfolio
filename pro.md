# 个人博客/作品集网站 - 产品需求文档

## 1. 项目概述
本项目是一个面向数字经济专业学生Mei的个人博客/作品集网站，用于展示个人信息、技能、项目成果和联系方式，同时提供简单的内容管理功能。

## 2. 项目目标
- 打造一个简洁、现代、响应式的个人品牌展示平台
- 提供清晰的个人信息和技能展示
- 展示项目成果并提供外部链接访问
- 支持简单的内容更新方式
- 实现特色视觉效果提升用户体验

## 3. 用户角色与核心功能

### 3.1 用户角色
- **访客 (HR/面试官)**: 快速了解Mei的专业背景、技术栈、项目经验，获取联系方式
- **访客 (技术同行)**: 浏览项目细节，通过GitHub链接进行技术交流
- **管理员 (Mei本人)**: 通过简单修改文件即可更新网站内容，无需复杂的技术操作

### 3.2 核心功能模块

| 模块名称 | 功能描述 | 技术难点/亮点 |
|---------|---------|-------------|
| 首页 (Hero) | 展示个人标语、简短介绍，集成粒子效果背景 | 鼠标引力+弹性连线粒子效果，响应式设计 |
| 关于我 (About Me) | 展示个人详细信息、照片、教育背景 | 响应式布局，图片自适应显示 |
| 技能展示 (Skills) | 分类展示专业技能和软件工具掌握情况 | 技能卡片悬停效果，图标可视化展示 |
| 项目展示 (Projects) | 卡片式布局展示项目成果，包含项目描述和外部链接 | CSS Grid响应式布局，悬停动画效果 |
| 相册展示 (Gallery) | 展示个人照片，每张照片配有文件名作为描述 | 瀑布流布局，图片懒加载 |
| 联系我 (Contact) | 展示联系方式，提供简单的联系表单 | 表单验证，响应式设计 |
| 全局功能 | 平滑滚动、呼吸式主题色、响应式导航 | 呼吸式主题色动画，移动端适配 |

## 4. 技术栈

### 4.1 前端技术
- **HTML5/CSS3**: 页面结构和样式设计
- **JavaScript**: 交互逻辑和动画效果
- **Canvas API**: 实现粒子效果
- **Font Awesome**: 图标库
- **Inter Font**: 现代无衬线字体

### 4.2 后端技术
- **静态网站**: 无需复杂后端，通过GitHub Pages部署

### 4.3 开发工具
- **Git**: 版本控制
- **GitHub**: 代码托管和项目部署

## 5. 界面设计要求

### 5.1 整体风格
- 简洁、现代、清晰的设计语言
- 柔和的色调，呼吸式主题色变化
- 充足的留白，良好的视觉层次

### 5.2 响应式设计
- 适配移动端、平板和桌面设备
- 移动端导航优化（汉堡菜单）
- 图片和布局自适应不同屏幕尺寸

### 5.3 交互效果
- 平滑滚动效果
- 卡片悬停动画
- 粒子效果交互
- 呼吸式主题色变化

## 6. 内容管理
- 通过直接修改HTML/CSS/JavaScript文件更新网站内容
- 图片资源放置在images目录，通过文件名管理
- 项目信息直接在index.html中修改

## 7. 部署与维护
- 部署在GitHub Pages上
- 通过Git进行版本控制和更新
- 定期备份代码到GitHub仓库

## 8. 开发计划

### 8.1 第一阶段：基础框架搭建
- 页面结构设计
- 基础样式实现
- 响应式布局开发

### 8.2 第二阶段：核心功能开发
- 各模块内容填充
- 交互效果实现
- 粒子效果集成

### 8.3 第三阶段：优化与部署
- 性能优化
- 兼容性测试
- GitHub Pages部署

### 8.4 第四阶段：内容更新与维护
- 内容完善
- 功能调整
- 定期更新
