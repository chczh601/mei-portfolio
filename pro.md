# 个人博客/作品集网站 - 产品需求文档

## 1. 项目概述
本项目是一个面向数字经济专业学生Mei的个人博客/作品集网站，用于展示个人信息、技能、项目成果和联系方式，同时提供简单的内容管理功能。

## 2. 项目目标
- 打造一个简洁、现代、响应式的个人品牌展示平台
- 提供清晰的个人信息和技能展示
- 展示项目成果并提供外部链接访问
- 支持简单的内容更新方式
- 实现特色视觉效果提升用户体验

## 3. 用户角色与核心功能

### 3.1 用户角色
- **访客 (HR/面试官)**: 快速了解Mei的专业背景（19岁，数字经济专业）、技术栈（Office、SQL、SPSS、数据分析/可视化）、项目经验（运势占卜网站、世界人文风景网站），获取联系方式
- **访客 (技术同行)**: 浏览项目细节，通过GitHub链接进行技术交流，查看技能展示和博客内容
- **管理员 (Mei本人)**: 通过直接修改HTML/CSS/JavaScript文件更新网站内容，无需复杂的技术操作

### 3.2 核心功能模块

| 模块名称 | 功能描述 | 技术难点/亮点 |
|---------|---------|-------------|
| 首页 (Hero) | 展示个人名称"Mei"、标语"Digital Economy Explorer"、简短介绍，集成粒子效果背景 | Canvas实现鼠标引力+弹性连线粒子效果，粒子组成"Mei"文字，响应式设计 |
| 关于我 (About Me) | 展示个人照片(Mei.jpg)、详细介绍、教育背景和个人信念 | 响应式布局，图片自适应显示，使用Inter字体增强可读性 |
| 技能展示 (Skills) | 分类展示7项专业技能（Office Suite、SQL、SPSS、数据分析、数据可视化、内容编辑、网页设计） | Font Awesome图标可视化展示，技能卡片悬停效果，CSS Grid响应式布局 |
| 项目展示 (Projects) | 卡片式布局展示2个项目成果：运势占卜网站和世界人文风景网站，包含项目描述和外部链接 | CSS Grid响应式布局，悬停动画效果，项目卡片使用Font Awesome图标增强视觉效果 |
| 博客 (Blog) | 展示3篇博客文章，包含标题、日期和简短描述 | 卡片式布局，悬停效果，响应式设计 |
| 联系我 (Contact) | 展示邮箱联系方式和社交媒体链接，提供联系表单 | 表单验证，响应式设计，使用Font Awesome社交图标 |
| 相册展示 (Gallery) | 通过JavaScript动态加载展示个人照片，每张照片配有文件名作为描述 | 动态图片加载，响应式网格布局，图片资源管理 |
| 全局功能 | 平滑滚动、呼吸式主题色（24秒循环）、响应式导航栏 | CSS `@property`实现呼吸式主题色动画，固定导航栏带有滚动效果，移动端适配 |

## 4. 技术栈

### 4.1 前端技术
- **HTML5**: 语义化页面结构，包含7个主要部分（首页、关于我、技能、项目、博客、联系我、相册）
- **CSS3**: 响应式样式设计，使用CSS Grid和Flexbox布局，实现呼吸式主题色动画（24秒循环）
- **JavaScript (ES6+)**: 实现粒子效果、动态相册加载、表单验证等交互功能
- **Canvas API**: 实现鼠标引力+弹性连线粒子效果，粒子组成"Mei"文字
- **Font Awesome 6.4.0**: 提供50+个图标用于技能、项目、联系表单和社交媒体
- **Inter Font**: 通过Google Fonts引入，作为网站主要字体，提升可读性

### 4.2 后端技术
- **静态网站**: 无需复杂后端，通过GitHub Pages部署，降低维护成本

### 4.3 开发工具
- **Git**: 版本控制，管理代码变更
- **GitHub**: 代码托管和项目部署（仓库地址：https://github.com/chczh601/mei-portfolio.git）
- **GitHub Pages**: 静态网站部署平台，提供免费的域名和托管服务

## 5. 界面设计要求

### 5.1 整体风格
- 简洁、现代、清晰的设计语言，以用户体验为中心
- 柔和的色调，使用#B6DDDC作为主色调，配合呼吸式主题色变化（24秒循环）
- 充足的留白，良好的视觉层次，使用Inter字体增强可读性
- 导航栏固定在顶部，带有半透明背景和毛玻璃效果，提升视觉层次感

### 5.2 响应式设计
- 适配移动端、平板和桌面设备，使用媒体查询实现不同屏幕尺寸的优化
- 布局使用CSS Grid和Flexbox，确保在各种设备上都能良好展示
- 图片和内容区块自适应不同屏幕尺寸，保持良好的视觉比例

### 5.3 交互效果
- 平滑滚动效果，提升页面浏览体验
- 卡片悬停动画（技能、项目、博客卡片），增强用户交互感
- 粒子效果交互（鼠标引力+弹性连线），粒子组成"Mei"文字，增加页面趣味性
- 呼吸式主题色变化（24秒循环），使用CSS `@property`实现流畅的色彩过渡
- 导航栏滚动效果，滚动时导航栏背景透明度增加，提升可用性

## 6. 内容管理
- **文本内容更新**: 通过直接修改index.html文件更新网站所有文本内容，包括个人介绍、技能描述、项目信息、博客文章等
- **图片资源管理**: 图片资源放置在images目录，使用清晰的文件名管理（如Mei.jpg作为个人照片）
- **相册内容**: 通过JavaScript动态加载images目录中的图片，每张照片自动使用文件名作为描述
- **样式调整**: 通过修改css/style.css文件调整网站样式，包括颜色、布局、动画效果等
- **功能扩展**: 通过修改js目录下的JavaScript文件（main.js、particles.js）扩展网站功能和交互效果

## 7. 部署与维护
- **部署平台**: 使用GitHub Pages部署网站，仓库地址：https://github.com/chczh601/mei-portfolio.git
- **部署流程**: 
  1. 使用Git将代码提交到GitHub仓库的main分支
  2. GitHub Pages自动检测代码变更并重新构建部署
  3. 可通过修改README.md文件触发缓存刷新，解决部署版本不更新问题
- **版本控制**: 使用Git进行代码版本控制，记录所有变更历史
- **定期备份**: 代码自动备份到GitHub仓库，确保数据安全
- **维护周期**: 定期检查网站可用性，根据需要更新内容和功能
