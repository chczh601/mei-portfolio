1. 浏览器标签页 Favicon 动画 

概念：把个人 Logo 做成逐帧动画，用户切换标签时 Favicon 会眨眼、放电或打 Morse code。 

技术： 

- 用 16×16 的 GIF 逐帧图 + 动态 `<link rel="icon" href="favicon_01.png">` 轮询； 

- 或 Canvas 离屏渲染 16×16 实时生成 base64 直接塞进 `<link>`。 

亮点：0 成本“彩蛋”，别人截屏都截不到。 

2. “呼吸式”主题色 

概念：整站只有一套 HSL 色值，但通过 CSS `@property` 把 H 轴（色相）绑定到 `requestAnimationFrame`，让站点色调像呼吸一样缓慢漂移，24 秒一圈。 

技术： 

```css 

@property --hue { 

syntax: '<number>'; 

inherits: false; 

initial-value: 0; 

} 

body { 

--hue: 0; 

background: hsl(var(--hue) 70% 96%); 

animation: hueRotate 24s linear infinite; 

} 

@keyframes hueRotate { to { --hue: 360; } } 

``` 

亮点：单色不花哨，却永远看不腻；打印或截图时自动落在某一帧，不影响可读性。 

3. 滚动“时间戳”简历 

概念：把工作经历做成垂直时间轴，但滚动时右侧出现“真实时间戳水印”，例如 2018.06.11 14:32，用 WebGL 着色器做成扫描线老终端效果。 

技术： 

- 用 Three.js 把文字纹理贴到 PlaneGeometry，随 `scrollY` 移动 UV； 

- 水印只在滚动时显现，停 1.2 s 后淡出，兼顾可读性与炫技。 

4. 访问者“共同创作”背景 

概念：背景是一张无限画布，所有访客的光标轨迹被 WebSocket 收集并实时广播，形成“幽灵之手”共同涂鸦；每人离开前可一键保存为 4K 壁纸并弹窗致谢。 

技术： 

- 后端用 Node + ws，房间按日期分片，防止爆炸； 

- 前端 Canvas2D 分层：本地轨迹 60 fps，网络轨迹 30 fps，降低延迟； 

- 用 SSE 推送当前在线人数，制造“并肩”氛围。 

注意：GDPR 提示 + 匿名化，避免存 IP。 

5. 字体“自我修复” 

概念：首屏标题每个字母被 `clip-path` 随机“碎裂”，滚动到视口时通过 SVG 路径变形动画自动“长回去”。 

技术： 

- 用 Greensock (GSAP) 的 `MotionPathPlugin` 把碎裂路径与原始路径做 `morphSVG`； 

- 文字仍是真实 DOM，SEO 无损； 

- 碎裂算法：把字母转成 SVG 路径后，用 `paper.js` 做 Voronoi 切分导出 `clip-path` polygon。 

6. 夜间“月光”模式 

概念：不是简单黑底白字，而是根据用户本地时间计算月相，用 CSS mask 把一张超大月亮 PNG 做成镂空蒙版，月光部分亮度 1.2 倍，其余区域压暗。 

技术： 

- 月相算法：用 2000 年简化版 Meeus 公式，压缩到 400 byte 的 JS； 

- `mask-image` 加 `backdrop-filter: brightness(0.8)`，性能比 WebGL 方案好。 

7. 文章“量子高亮” 

概念：选中一段文字后，页面会生成 N（=选中字数）个 1×1 的 div，每个 div 颜色取自该字符在 Unicode 表中的序号映射到 HSL，形成“文字即调色盘”效果。 

技术： 

- `getSelection().toString()` 拆字符，动态创建绝对定位 div，用 `top/left` 跟随原字符坐标（Range.getBoundingClientRect 每字级别）； 

- 关闭弹窗或点击空白时，div 做 Poisson-disk 爆炸飞散，再 `remove()`。 

8. 404 “故障艺术”自画像 

概念：404 页面先显示作者正常 SVG 线稿，随后用 WebGL 后期通道模拟 CRT 抖动、色偏、撕裂，最终冻结成一张“蓝屏”截图，按钮“重启我”返回首页。 

技术： 

- 用 Three.js 后期通道 `EffectComposer` + 自定义 glitch pass； 

- 冻结瞬间把 Canvas 快照到 CSS `background-image`，隐藏 WebGL 画布，减少后续耗电。 

9. 只用 CSS 实现“声音可视化” 

概念：没有 Web Audio，也不放配乐，而是把作者事先导出的 44 kHz 波形 JSON 拆成 128 段，预生成 128 个 CSS keyframe 动画，类名 `.b0 ~ .b127`，通过 `animation-delay` 模拟音条跳动。 

亮点：纯 CSS 0 JS 运行时成本，适合放在“关于我”背景，关掉动画偏好即自动暂停。 

10. 打印样式“彩蛋” 

概念：按 `Ctrl+P` 时，页面自动把当前文章排版成一本 A5 zine（杂志）样式： 

- 标题变成手写体 SVG 盖章； 

- 所有外链转成 QR 小图标附在段尾； 

- 结尾附一张“打印时间”热敏票据，用 `@media print` 实现。 

技术： 

- 用 `window.matchMedia('print')` 监听，提前把 `<link rel="stylesheet" media="print">` 注入； 

- QR 用 `qrcode-svg` 现算，不污染正常浏览性能。 

落地建议 

1. 先选 1–2 个“大创意”做主轴，其余做彩蛋，避免“全家桶”导致首屏 5 MB。 

2. 把花哨效果全部放到 `prefers-reduced-motion: no-preference` 媒体查询内，尊重可访问性。 

3. 用 `partytown` 把 WebGL/Heavy 脚本丢到 web worker，不阻塞主线程。 

4. 所有需要后端的功能（共同创作、WebSocket）都先做匿名化 + 速率限制，防止被撸爆。

2. 烟花“自爆”404 页 

- 效果 

进入 404 后，所有文字先碎裂成 2000 个像素粒子向上喷射，做重力抛物线，最后缓缓下落熄灭；点击屏幕可重播。 

- 实现 

纯 Canvas2D，核心 80 行： 

```js 

class P { 

constructor(x,y,color){ 

this.x=x;this.y=y; 

this.vx=(Math.random()-0.5)*8; 

this.vy=Math.random()*-12-4; 

this.life=1;this.color=color; 

} 

update(){this.vy+=0.3;this.x+=this.vx;this.y+=this.vy;this.life-=0.015;} 

} 

``` 

完整教程与源码：CSDN《HTML5 粒子动态背景特效》。 

- 适合：404 彩蛋，仪式感拉满。 

3. 文字 → 粒子“消散/重组”Banner
